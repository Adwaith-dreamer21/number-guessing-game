# number-guessing-game

A simple and interactive **Number Guessing Game** built using Python.  
The program randomly selects a number within a specified range, and the player must guess the correct number with the help of hints provided after each attempt.

# Features
- Generates a random number within a defined range  
- Provides hints: **"Too High"**, **"Too Low"**, or **"Correct!"**  
- Simple and beginner-friendly logic  
- Runs in any Python-supported environment  
- Great practice project for loops, conditionals, and input handling
